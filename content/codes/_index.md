# Codes

## LaTeX tools & templates

<span style="font-size:.9em;">

**Lazy LaTeX** (for VSCode)\
Collections of LaTeX snippets for Visual Studio Code, extending [LaTeX Workshop](https://github.com/James-Yu/LaTeX-Workshop) by James Yu.\
Download the latest version on <a href="https://gitHub.com/MartinDbx/Lazy-LaTeX">GitHub</a>.

**UMons Metropolis**\
An unofficial but well received UMons template for the beamer class, based on <a href="https://github.com/matze/mtheme">metropolis</a>.\
Download the latest version on <a href="https://github.com/MartinDbx/umons-metropolis">GitHub</a>.

</span>

## Red Dead Online tools

<span style="font-size:.9em;">

**RDO - Collector's Bag to Map**\
Automatic settings file of your uncollected items for jeanropke’s <a href="https://jeanropke.github.io/RDR2CollectorsMap/">Collectors Map</a>.\
Download the latest version on <a href="https://github.com/MartinDbx/RDO-CollectorsBagToMap">GitHub</a>.

**RDO - Collector's Shop to Map**\
Custom settings file of your uncollected items for the jeanropke's <a href="https://jeanropke.github.io/RDR2CollectorsMap/">Collectors Map</a>.\
Download the latest version on <a href="https://github.com/MartinDbx/RDO-CollectorsShopToMap">GitHub</a>.

</span>
