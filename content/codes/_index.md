# Codes

## LaTeX tools & templates

<span style="font-size:.9em;">

**Lazy LaTeX** (for VSCode)\
Collections of LaTeX snippets for Visual Studio Code, extending [LaTeX Workshop](https://github.com/James-Yu/LaTeX-Workshop) by James Yu.\
Download the latest version on [GitHub](https://github.com/MartinDbx/Lazy-LaTeX).

**UMons Metropolis**\
An unofficial (but well received) UMons template for the beamer class. Based on the [metropolis](https://github.com/matze/mtheme) beamer theme.\
Download the latest version on [GitHub](https://github.com/MartinDbx/umons-metropolis).

</span>

## Red Dead Online tools

<span style="font-size:.9em;">

**RDO - Collector's Bag to Map**\
Automatic settings file of your uncollected items for jeanropke’s [Collectors Map](https://jeanropke.github.io/RDR2CollectorsMap/).\
Download the latest version on [GitHub](https://github.com/MartinDbx/Lazy-LaTeX).

**RDO - Collector's Shop to Map**\
Custom settings file of your uncollected items for the jeanropke's [Collectors Map](https://jeanropke.github.io/RDR2CollectorsMap/).\
Download the latest version on [GitHub](https://github.com/MartinDbx/RDO-CollectorsShopToMap).

</span>
