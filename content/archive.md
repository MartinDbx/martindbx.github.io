# Archive


<span style="font-size:.9em;">

**Xmath** LaTeX package\
Extension of frequently used mathematical packages with new commands for specific sets, arrows and operators. This package also implements useful shortcuts.\
<span style="color:red;">Depreciated</span> • Stored on <a href="https://github.com/MartinDbx/xmath">GitHub</a>.

**Xwriter** LaTeX package\
Simple way to structure a consistent layout, by defining a common framework which supports different typographic styles. Also includes a large selection of mathematical environments.\
<span style="color:red;">Depreciated</span> • Stored on <a href="https://github.com/MartinDbx/xmath">GitHub</a>.
